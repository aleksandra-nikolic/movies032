Movies032

A responsive web application for browsing movies and TV shows, built with Angular.
Data is fetched from an external API and includes a search feature and detailed pages for each title.

Features
- Search movies and TV shows
- Detailed page for each movie/TV show with description
- Responsive design (mobile friendly)

Technologies
- Angular
- PrimeNG
- Tailwind CSS
- REST API
- [TMDB API](https://www.themoviedb.org/documentation/api)

Run locally
1. Clone the repository
   git clone https://github.com/aleksandranikolic96/movies032.git
2. Install dependencies
   npm install
3. Start the development server
   ng serve
4. Open http://localhost:4200 in your browser

Note
This app uses the [TMDB API](https://www.themoviedb.org/documentation/api).
This product uses the TMDB API but is not endorsed or certified by TMDB.
